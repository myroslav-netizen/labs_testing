# Visual Studio Code

Visual Studio Code є легким, але потужним редактором коду, який підтримує широкий спектр мов програмування. Він пропонує:
- зручну інтеграцію з Git, 
- розширеннями,
- інструментами для налагодження. 
### Завдяки гнучкому інтерфейсу та великій кількості плагінів VS Code підходить як для новачків, так і для досвідчених розробників.
